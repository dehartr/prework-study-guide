HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • In semantic HTML elements, the name of the element describes its content and the role it plays on a webpage.
  • The meta elements contain information about the page that is used by the browser.
  • The user won't see information in the head or meta elements
  • The h1, h2, h3, h4, h5 and h6 elements represent the level of heading a given text block represents.
  • The p element represents a paragraph or block of text. You'll use this element extensively to wrap most of the text on your webpages.

CSS

  • CSS Cascading Style Sheets
  • programming language used by developers to (style the webpage) define how webpage content is presented to users/li>
  • CSS defines the colors, aesthetics, visual layout, and organization.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • inline CSS is used to sytle an element directly in the HTML file and must be edited indiviually whenever you make a design change
  • Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only.
  • internal style sheet contains CSS rules for the webpage in the head section of the HTML file.
  • an external CSS style sheet is the most common way to style a webpage is by using
  • it allows developers to keep all their CSS rules in a separate file, which makes design changes easier.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
